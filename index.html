Prototype Konverter Arab Braille Indonesia

Tekan tombol Convert untuk mengkonversi

Fitur:

  • Support text Quran dengan script Utsmani dan standarKemenag.
  • Penulisan bentuk mad mengikuti ketentuan penulisan Braille di Indonesia.
  • Deteksi huruf alif wassal pada Quran Kemenag untuk menghindari kesalahan penulisan bentuk mad.
  • Untuk Quran standar Kemenag, huruf alif yang diberi tanda harakat akan diubah menjadi Hamzah 'alal alif dengan harakat, mengikuti aturan baku tulisan arab.

Catatan:

  • Belum support tanda Waqaf.
  • Masih terdapat beberapa simbol yang belum terdata.
  • Simbol yang tidak terdata akan ditampilkan sebagaimana bentuk aslinya.
  • Jika dinilai potensial, proyek ini akan terus dikembangkan, baik dalam bentuk website, atau bentuk program lainnya.

The scripts are written using pure Javascript, HTML, and CSS without any dependencies or frameworks.
